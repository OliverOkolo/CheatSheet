HTML Cheatsheet

Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language used to create web pages. Below is a detailed cheatsheet of the most commonly used HTML tags and their descriptions.

Basic Structure Tags

Tag Description Example
<html></html> Defines the root of an HTML document. <html lang="en"></html>
<head></head> Contains metadata and links to external resources. <head></head>
<body></body> Defines the content of the document. <body></body>

Text Formatting Tags

Tag Description Example
<p></p> Defines a paragraph. <p>This is a paragraph.</p>
<h1> to <h6> Defines headings, from most important (<h1>) to least (<h6>). <h1>Main Heading</h1>
<strong></strong> Makes text bold. <strong>Bold Text</strong>
<em></em> Makes text italic. <em>Italic Text</em>

Links and Media Tags

Tag Description Example
<a href=""></a> Creates a hyperlink. <a href="https://example.com">Click here</a>
<img src=""> Embeds an image. <img src="image.jpg" alt="Description">
<video></video> Embeds a video. <video src="video.mp4" controls></video>
<audio></audio> Embeds audio content. <audio src="audio.mp3" controls></audio>

Forms and Input Tags

HTML forms collect user input. Below are common tags used in forms:

Tag Description Example
<form></form> Creates a form element. <form action="/submit"></form>
<input> Defines an input field. <input type="text" name="username">
<textarea></textarea> Defines a multi-line text input. <textarea rows="4" cols="50"></textarea>
<button></button> Defines a clickable button. <button type="submit">Submit</button>
